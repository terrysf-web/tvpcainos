Ainos V3 Cache Reset

중요:
업로드 후 화면 왼쪽 위에 빨간 배지로
"V3 CACHE RESET LOADED"
가 보여야 새 파일이 실제로 로드된 것입니다.

이 배지가 안 보이면:
- GitHub Pages가 아직 새 파일을 배포하지 않았거나
- iPad 홈화면 앱/Safari 캐시가 예전 파일을 보고 있는 것입니다.

적용:
1. ZIP 압축 풀기
2. tvpcainos repo root에 모든 파일 업로드/교체
3. GitHub Pages 배포 완료 후
4. iPad Safari에서 주소 뒤에 ?v=3reset 붙여서 열기:
   https://terrysf-web.github.io/tvpcainos/index.html?v=3reset
5. 빨간 V3 CACHE RESET LOADED 배지 확인
6. 그 다음 홈화면 앱도 삭제 후 다시 추가 권장
