TVPC Worship Charts - Buildup v1

적용 방법:
1) ZIP 압축을 풉니다.
2) index.html, manifest.json, icon-192.png, icon-512.png 파일을 GitHub Pages repo에 업로드/교체합니다.
3) iPad에서 새로고침 후 홈 화면 앱으로 실행합니다.

v1 핵심:
- Service 화면: 악보만 fullscreen, Crop/Zoom/Save/Reset 버튼 없음
- Song Library 화면: Crop/Edit 가능
- 곡별 crop/zoom 저장
- Setlist swipe 이동
- 첫 곡에서 이전 swipe 시 아무 동작 없음
- Share URL은 https://terrysf-web.github.io/tvpc-worship/index.html#share=... 로 고정

주의:
- PDF는 각 iPad 브라우저 내부 IndexedDB에 저장됩니다.
- 공유 링크는 setlist/song 정보 공유용이며 PDF 파일 자체는 공유하지 않습니다.
